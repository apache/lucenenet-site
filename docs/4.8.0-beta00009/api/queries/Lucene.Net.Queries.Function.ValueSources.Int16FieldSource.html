    Class Int16FieldSource

    Obtains System.Int16 field values from the FieldCache using GetInt16s(AtomicReader, String, FieldCache.IInt16Parser, Boolean) and makes those values available as other numeric types, casting as needed.

    NOTE: This was ShortFieldSource in Lucene
