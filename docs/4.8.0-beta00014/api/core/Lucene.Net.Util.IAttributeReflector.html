    Reflect(Type, String, Object)

    This method gets called for every property in an Attribute/AttributeSource passing the System.Type of the IAttribute, a key and the actual value. E.g., an invocation of ReflectWith(IAttributeReflector) would call this method once using typeof(Analysis.TokenAttributes.ICharTermAttribute) as attribute type, "term" as key and the actual value as a System.String.
