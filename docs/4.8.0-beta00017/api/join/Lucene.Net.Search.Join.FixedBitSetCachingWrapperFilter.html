    Methods

    DocIdSetToCache(DocIdSet, AtomicReader)

    Provide the Lucene.Net.Search.DocIdSet to be cached, using the Lucene.Net.Search.DocIdSet provided by the wrapped Filter.

    This implementation returns the given Lucene.Net.Search.DocIdSet, if Lucene.Net.Search.DocIdSet.IsCacheable returns true, else it calls Lucene.Net.Search.CachingWrapperFilter.CacheImpl(Lucene.Net.Search.DocIdSetIterator, Lucene.Net.Index.AtomicReader)

    Note: this method returns Lucene.Net.Search.CachingWrapperFilter.EMPTY_DOCIDSET if the given docIdSet is null or if Lucene.Net.Search.DocIdSet.GetIterator() return null. The empty instance is use as a placeholder in the cache instead of the null value.
