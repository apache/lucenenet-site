    Class ArabicStemmer

    Stemmer for Arabic.

    Stemming is done in-place for efficiency, operating on a termbuffer.

    Stemming is defined as:
    • Removal of attached definite article, conjunction, and prepositions.
    • Stemming of common suffixes.
