    GetScorer(AtomicReaderContext, IBits)

    Returns a Lucene.Net.Search.Scorer which scores documents in/out-of order according to scoreDocsInOrder.

    NOTE: even if scoreDocsInOrder is false, it is recommended to check whether the returned Lucene.Net.Search.Scorer indeed scores documents out of order (i.e., call Lucene.Net.Search.Weight.ScoresDocsOutOfOrder), as some Lucene.Net.Search.Scorer implementations will always return documents in-order.

    NOTE:null can be returned if no documents will be scored by this query.
