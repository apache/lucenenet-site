    OpenIndexWriter(Directory, IndexWriterConfig)

    Open internal index writer, which contains the taxonomy data.

    Extensions may provide their own Lucene.Net.Index.IndexWriter implementation or instance.

    NOTE: the instance this method returns will be disposed upon calling to Lucene.Net.Facet.Taxonomy.Directory.DirectoryTaxonomyWriter.Dispose().

    NOTE: the merge policy in effect must not merge none adjacent segments. See comment in Lucene.Net.Facet.Taxonomy.Directory.DirectoryTaxonomyIndexWriterFactory.CreateIndexWriterConfig(Lucene.Net.Index.OpenMode) for the logic behind this.
