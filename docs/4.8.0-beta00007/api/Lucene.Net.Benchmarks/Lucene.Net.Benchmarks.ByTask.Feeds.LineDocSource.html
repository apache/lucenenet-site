    Class LineDocSource

    A ContentSource reading one line at a time as a Document from a single file. This saves IO cost (over DirContentSource) of recursing through a directory and opening a new file for every document.

    Remarks

    The expected format of each line is (arguments are separated by <TAB>): title, date, body. If a line is read in a different format, a will be thrown. In general, you should use this content source for files that were created with WriteLineDocTask.
