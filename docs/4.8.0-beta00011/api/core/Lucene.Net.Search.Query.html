    Class Query

    The abstract base class for queries.

    Instantiable subclasses are:

    • TermQuery
    • BooleanQuery
    • WildcardQuery
    • PhraseQuery
    • PrefixQuery
    • MultiPhraseQuery
    • FuzzyQuery
    • RegexpQuery
    • TermRangeQuery
    • NumericRangeQuery
    • ConstantScoreQuery
    • DisjunctionMaxQuery
    • MatchAllDocsQuery

    See also the family of Span Queries (Lucene.Net.Search.Spans) and additional queries available in the Queries module
