    Class CrankyTokenFilter

    Throws System.IO.IOException from random Lucene.Net.Analysis.TokenStream methods.

    This can be used to simulate a buggy analyzer in Lucene.Net.Index.IndexWriter, where we must delete the document but not abort everything in the buffer.
