    Class SpatialDocMaker

    Indexes spatial data according to a configured Lucene.Net.Spatial.SpatialStrategy with optional shape transformation via a configured IShapeConverter. The converter can turn points into circles and bounding boxes, in order to vary the type of indexing performance tests. Unless it's subclass-ed to do otherwise, this class configures a Spatial4n.Core.Context.SpatialContext, Lucene.Net.Spatial.Prefix.Tree.SpatialPrefixTree, and Lucene.Net.Spatial.Prefix.RecursivePrefixTreeStrategy. The Strategy is made available to a query maker via the static method GetSpatialStrategy(Int32). See spatial.alg for a listing of spatial parameters, in particular those starting with "spatial." and "doc.spatial".
