    Class FrenchStemFilter

    A Lucene.Net.Analysis.TokenFilter that stems french words.

    The used stemmer can be changed at runtime after the filter object is created (as long as it is a FrenchStemmer).

    To prevent terms from being stemmed use an instance of KeywordMarkerFilter or a custom Lucene.Net.Analysis.TokenFilter that sets the KeywordAttribute before this Lucene.Net.Analysis.TokenStream.
