    Tags

    Gets or sets the POS tag of the term. A single word may have multiple POS tags, depending on the interpretation (context disambiguation is typically needed to determine which particular tag is appropriate).

    The default value (no-value) is null. Returns a list of POS tags corresponding to current lemma.
